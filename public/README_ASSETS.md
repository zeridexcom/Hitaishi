# PWA assets

This directory contains the static assets for the Hitaishi PWA.

## What's here

- `manifest.json` — Web app manifest, referenced from `app/layout.tsx`.
- `hitaishi-logo.svg` — Master brand mark (vector). Use this as the source for icon exports.
- `icon-192.png` — 192×192 raster icon. Required for PWA install.
- `icon-512.png` — 512×512 raster icon. Required for PWA install (also doubles as the `maskable` variant — see manifest).
- `apple-touch-icon.png` — 180×180 icon for iOS "Add to Home Screen".

## Placeholders

The PNGs currently committed are **flat-color placeholders** generated by
`scripts/write-placeholder-icons.mjs` (pure-Node, no `sharp` dependency) so
the manifest doesn't 404 in the meantime.

Before launch, replace them with real assets:

```bash
# 1. install sharp (one-time)
npm i -D sharp

# 2. regenerate the PNGs from the SVG
node scripts/generate-pwa-icons.mjs
```

`scripts/generate-pwa-icons.mjs` rasterizes `hitaishi-logo.svg` at 192×192,
512×512, and 180×180, then writes the three files in this directory.

If you change the brand mark, edit `hitaishi-logo.svg` first, then re-run
the script — there is no separate "source PNG" to keep in sync.

## Brand colors

Primary: `#0b6445` (deep) / `#2f7d5c` (default)
Accent:  `#e87b5a` (coral)
Surface: `#f7faf5` (background) / `#ffffff` (cards)
Ink:     `#181d1a` (default) / `#3f4943` (soft) / `#6f7a72` (faint)
